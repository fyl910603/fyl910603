 作为函数时，super()只能在子类的构造函数之中，用在其它地方会报错

<pre>`     <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">A</span>{</span>}

#### super作为对象时

<pre>`     <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">A</span>{</span>
        p(){
            <span class="hljs-keyword">return</span> <span class="hljs-number">2</span>;
        }
    }
    <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">B</span> <span class="hljs-keyword"><span class="hljs-keyword">extends</span></span> <span class="hljs-title">A</span>{</span>
        constructor(){
            <span class="hljs-keyword">super</span>();
            console.log(<span class="hljs-keyword">super</span>.p())
        }
    }
    let b = <span class="hljs-keyword">new</span> <span class="hljs-type">B</span>();
`</pre>

 super在普通方法中，指向A.prototype,super.p()相当于A.prototype.p()
 由于super指向父类的原型对象，所以定义在父类实例上的方法或属性无法通过super调用

<pre>`     <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">A</span>{</span>
        constructor(){
            <span class="hljs-keyword">this</span>.p = <span class="hljs-number">2</span>;
        }
    }
    <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">B</span> <span class="hljs-keyword"><span class="hljs-keyword">extends</span></span> <span class="hljs-title">A</span>{</span>
        get m(){
            <span class="hljs-keyword">return</span> <span class="hljs-keyword">super</span>.p;
        }
    }
    let b = <span class="hljs-keyword">new</span> <span class="hljs-type">B</span>();
    b.m <span class="hljs-comment">//undefined</span>
`</pre>

 以上，p是父类A实例的属性，super.p引用不到
 如果属性定义在父类的原型对象上，super就可以取到

<pre>`     <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">A</span>{</span>}
    <span class="hljs-type">A</span>.prototype.p = <span class="hljs-number">2</span>;
    <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">B</span> <span class="hljs-keyword"><span class="hljs-keyword">extends</span></span> <span class="hljs-title">A</span>{</span>
        constructor(){
            <span class="hljs-keyword">super</span>();
            console.log(<span class="hljs-keyword">super</span>.p)
        }    
    }
    let b = <span class="hljs-keyword">new</span> <span class="hljs-type">B</span>();
`</pre>

 ES6规定，通过super调用父类的方法时，super会绑定子类的this

<pre>`     <span class="hljs-keyword">class</span> A{
        <span class="hljs-constructor"><span class="hljs-keyword">constructor</span>()</span>{
            <span class="hljs-keyword">this</span>.x = <span class="hljs-number">1</span>;
        }
        print(){
            <span class="hljs-built_in">console</span>.log(<span class="hljs-keyword">this</span>.x);
        }
    }
    <span class="hljs-keyword">class</span> B extends A{
        <span class="hljs-constructor"><span class="hljs-keyword">constructor</span>()</span>{
            <span class="hljs-keyword">super</span>();
            <span class="hljs-keyword">this</span>.x = <span class="hljs-number">2</span>;
        }
        m(){
            <span class="hljs-keyword">super</span>.print();
        }
    }
    <span class="hljs-keyword">let</span> b = <span class="hljs-keyword">new</span> B();
    b.m();<span class="hljs-comment">//2</span>
`</pre>

 A.prototype.print()绑定了子类B的this
 如果通过super对某个属性赋值，这是super就是this，赋值的属性会变成子类实例的属性

<pre>`     <span class="hljs-keyword">class</span> A{
        <span class="hljs-constructor"><span class="hljs-keyword">constructor</span>()</span>{
            <span class="hljs-keyword">this</span>.x = <span class="hljs-number">1</span>;
        }
    }
    <span class="hljs-keyword">class</span> B extends A{
        <span class="hljs-constructor"><span class="hljs-keyword">constructor</span>()</span>{
            <span class="hljs-keyword">super</span>();
            <span class="hljs-keyword">this</span>.x = <span class="hljs-number">2</span>;
            <span class="hljs-keyword">super</span>.x = <span class="hljs-number">3</span>;
            <span class="hljs-built_in">console</span>.log(<span class="hljs-keyword">super</span>.x);
            <span class="hljs-built_in">console</span>.log(<span class="hljs-keyword">this</span>.x);
        }
    }
    <span class="hljs-keyword">let</span> b = <span class="hljs-keyword">new</span> B();
`</pre>

 super.x赋值为3等同于this.x赋值为3，而当输出super.x的时候读取的是A.prototype.x,因此返回undefined

 如果super作为对象，用在静态方法中，这是super将指向父类，而不是父类的原型对象。

<pre>`     <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">Parent</span>{</span>
        static myMethod(msg){
            console.log(<span class="hljs-symbol">'stati</span>c',msg);
        }
        myMethod(msg){
            console.log(<span class="hljs-symbol">'instanc</span>e',msg);
        }
    }
    <span class="hljs-class"><span class="hljs-keyword">class</span> <span class="hljs-title">Child</span> <span class="hljs-keyword"><span class="hljs-keyword">extends</span></span> <span class="hljs-title">Parent</span>{</span>
        static myMethod(msg){
            <span class="hljs-keyword">super</span>.myMethod(msg);
        }
        myMethod(msg){
            <span class="hljs-keyword">super</span>.myMethod(msg);
        }
    }
    <span class="hljs-type">Child</span>.myMethod(<span class="hljs-number">1</span>);<span class="hljs-comment">//static 1</span>
    <span class="hljs-keyword">var</span> child = <span class="hljs-keyword">new</span> <span class="hljs-type">Child</span>();
    child.myMethod(<span class="hljs-number">2</span>);/instance <span class="hljs-number">2</span>
`</pre>

 super在静态方法之中指向父类，在普通方法只能怪指向父类的原型对象
注意：使用super的时候必须**显式指定作为函数还是对象使用**

<pre>`    <span class="hljs-keyword">class</span> A{}
    <span class="hljs-keyword">class</span> B extends A{
        <span class="hljs-constructor"><span class="hljs-keyword">constructor</span>()</span>{
            <span class="hljs-keyword">super</span>();
            <span class="hljs-built_in">console</span>.log(<span class="hljs-keyword">super</span>.valueOf() <span class="hljs-keyword">instanceof</span> B);<span class="hljs-comment">//true</span>
        }
    }
    <span class="hljs-keyword">let</span> b = <span class="hljs-keyword">new</span> B();
`</pre>

 super.valueOf()表明super是一个对象，同时super绑定B的this，所以super.valueOf()返回一个B的实例  

 由于对象总是继承其它对象，因此可在任何一个对象中使用super关键字

    }
 };
 obj.<span class="hljs-built_in">toString</span>();<span class="hljs-comment">//</span>
